Analyzing Unicorn Companies

Tech Stack: PostgresSQL(CTEs, JOINs, Aggregations)

This SQL analysis identifies the top 3 industries that produced the most unicorn startups between 2019 and 2021. The project uses multiple CTEs to filter and summarize data from a startup funding database, joining company, industry, and valuation tables.

Key Insights:

  • The top 3 industries with the most unicorns between 2019 and 2021 were consistently dominant in both count and valuation.
  • 2021 saw the highest spike in average unicorn valuation, suggesting a funding surge during the pandemic rebound.
  • Some industries had fewer unicorns but significantly higher average valuations, pointing to high investor confidence in specific sectors.

AI Integrated Inventory Management System (AI-IIMS)

Tech Stack: Python (Pandas, Prophet, FastAPI), Next.js, React.js, Tailwind CSS, Git/GitHub

AI-IIMS is a full-stack inventory forecasting solution developed in Agile sprints. The system forecasts demand for ingredients by analyzing historical sales data at the menu item level and translating those forecasts into ingredient quantities based on each recipe.

Deliverables:

  • Data ingestion and cleaning pipeline using Python and OpenPyXL
  • Forecasting engine using Prophet to predict daily demand for menu items and required ingredients
  • Recipe-to-ingredient mapping logic to calculate daily ingredient usage
  • RESTful API with FastAPI to handle user queries and serve predictions to the frontend
  • Interactive frontend using React and Tailwind CSS for input, selection, and visualization
  • Full project documentation such as; project scope, team charter, personas, user stories, meeting minutes, and final presentation
